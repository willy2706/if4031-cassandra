# Kumpulan perintah

1. Tambah user
```
register <username> <password>
```
2. Follow user
```
follow <username> <follower>
```
3. Tweet
```
tweet <username> <body>
```
4. Tampilkan tweet per user
```
displaytweet
```
5. Tampilkan timeline per user
```
displaytimeline
```
6. Keluar
```
exit
```
